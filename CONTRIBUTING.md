# Contributing to Kagemusha AI

Thank you for your interest in contributing to Kagemusha AI (影武者AI).

## Getting Started

See the [Development section in README.md](README.md#development) for setup instructions.

## Guidelines

- Format code with `pnpm exec dprint fmt` before committing
- Run `pnpm -r typecheck` after TypeScript changes
- Run `cargo check` after Rust changes
- Follow the branch naming convention: `fix/`, `chore/`, `refactor/` prefixes
- Keep Markdown-centric note experience as a priority
- Maintain local-first / privacy-first principles in all features
